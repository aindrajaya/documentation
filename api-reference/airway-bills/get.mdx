---
title: 'Get Airway Bills'
openapi: 'GET /api/airway-bills'
---

Retrieve a list of airway bills with optional filtering and pagination.

## Query Parameters

<ParamField query="page" type="integer" default="1">
  Page number for pagination
</ParamField>

<ParamField query="limit" type="integer" default="10">
  Number of items per page (max: 100)
</ParamField>

<ParamField query="awbNumber" type="string">
  Filter by airway bill number
</ParamField>

<ParamField query="shipper" type="string">
  Filter by shipper name
</ParamField>

<ParamField query="commodityCode" type="string">
  Filter by commodity code
</ParamField>
