---
title: 'Get Flights'
openapi: 'GET /api/flights'
---

Retrieve a list of flights with optional filtering and pagination.

## Query Parameters

<ParamField query="page" type="integer" default="1">
  Page number for pagination
</ParamField>

<ParamField query="limit" type="integer" default="10">
  Number of items per page (max: 100)
</ParamField>

<ParamField query="origin" type="string">
  Filter by origin airport code
</ParamField>

<ParamField query="destination" type="string">
  Filter by destination airport code
</ParamField>

<ParamField query="status" type="string">
  Filter by flight status (Scheduled, Boarding, In Transit, Arrived, Delayed, Cancelled)
</ParamField>

<ParamField query="date" type="string" format="date">
  Filter by flight date (YYYY-MM-DD)
</ParamField>
